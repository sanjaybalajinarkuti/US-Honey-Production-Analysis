# US-Honey-Production-Analysis

Analyzed 27 years (1995–2021) of US honey production data across 44 states using Python (Pandas, NumPy) — cleaning
1,115 records to ensure analysis-ready data; uncovered an 89% production decline from peak (220.3M lbs in 2000 →
23.2M lbs in 2021), a 305% price surge (1995–2017), and yield variations of 19–155 lbs/colony across states; and built
20+ visualizations (bar, pie, scatter, heatmap, violin, boxplot) using Matplotlib, Seaborn, and Plotly to communicate
regional and market insights
